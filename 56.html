Binance 월간 상승 패턴 비교 스캐너

여러 코인과 각기 다른 기준 월을 넣으면, 해당 월의 일봉 흐름을 바탕으로 공통 패턴을 자동 추출합니다. 단순 월간 상승률이 아니라 EMA / RSI / MACD / 볼린저밴드 / 거래량 배수 / 변동성을 함께 비교합니다.

지원 방식
여러 코인 + 각 코인별 다른 월
데이터 기준
Binance USDⓈ-M Futures 1D 캔들 (UTC)
결과 형태
공통점 요약 + 표 + 비교 차트

비교 샘플 입력

예시 기본값: FHE 2026-01 / POWER 2026-02
심볼은 FHE, FHEUSDT 둘 다 입력 가능하며, 월은 YYYY-MM 형식으로 입력하세요.
